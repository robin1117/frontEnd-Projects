The way elements are arranged on the webpage is known as document flow. Generally the default flow is from top to bottom or left to right on the basis of block or inline element -

Static

This is the default Value , and it means that the element will flow in the normal documents flow, according to other element around

Relative

Relative positioning is the position of an element relative to its normal position

Absolute

A element with the absolute position will move according to the position of it parent element in the absence of any parent element , the HTML element will places relative to the page

Fixed

An elemnt with position fixed will remain stuck to a specific position ever after the page is scroll

Sticky

It is a Hybrid of relative and fixed position, It becomes Sticky with in the parent
