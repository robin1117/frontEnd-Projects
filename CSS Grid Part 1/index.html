What is Grid ?

Grid Layout is a 2D layout System for Creating rows and columns. It allows for Creating Flexible and Responsive Layout that can adapt to different screen size and devices

We shall look the grid properties in two ways

  • Properties are to be applied on grid Container (Parent)
  • Properties are to be applied on grid Item (Chilld)

-->When display property is to :grid or :inline-grid the element becomes a grid container

Properties Associated with Grid Conainer ( Parent )

  • Grid-template-columns : with this property we can Explictely define the number of columns in the grid and width of each column. Here, there is a provision to use the specification as “auto” if all columns should have the same width
  • Grid-template-rows : It simply states the height of each row.
  • Align-content : vertically aligns the whole grid inside the specified container. It can be start, center, end, space evenly, space around, space between.
  • Justify-content : it aligns the whole grid inside the container. It can be start, center, space evenly, space around, space between.
  • Grid-auto-flow :Using this property we can define the flow of our grid-elements, and we set what would like to fill first columns or rows, its default value is :row
  • Grid-auto-column Using this we can define the width of the column which may be added automatically when we have more no of elements
  • Grid-auto-row Using this we can define the row of the column which may be added automatically
  • align-items: used to align grid element veritically with in the grid box
  • justify-items: used to align grid element hohrizontally with in the grid box
  • grid-template:short hand property to for setting grid-template-columns: grid-template-row: with in single line
    • grid-template:Rows/Colums;

We can use autofill or autofit and minmax() property inside repeat() for getting more responsiveness

Properties associated with grid items(Children)

  • Grid-column: its a short hand property for setting where the item is from start and end,you can either use line numbers, or use span to define how many columns the item will span.
    • Short Hand for ? grid-column-start: 1; grid-column-end: 2;
  • Grid-row: its a short hand property for setting where the item is from start and end,you can either use line numbers, or use span to define how many row the item will span.
    • Short Hand for ? grid-row-start: 1; grid-row-end: 2;
  • Grid-area: For the attributes grid-row-start, grid-column-start, grid-row-end, and grid-column-end, the grid-area property can be used as a shorthand.
