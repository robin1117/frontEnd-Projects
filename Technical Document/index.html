Introduction

Some Common Definition of OS ?

  • It is an interface between user and Hardware
  • It is resources manager
  • Set of utilities to simply application Development
  • Control program Software
  • Act like a Government

Functions of Control Unit

  • It Generate Timing or Control Signals in the form of clock Pulses, with theses it controls all other units .
  • It decides the Sequence execution of Micro-Operation

Functions of ALU

Arithmetic Logic Unit or Functional Unit is Responsible for doing tasks like ADDITION , SUBSCTRACTION , MUL , DIVISION etc

Together Control Unit and ALU we call it as a Processor / CPU / Micro Processor.

Primary Memory

  • it is also known Main/Physical Memory
  • Examples -> RAM, ROM, Cache , Registers
  • its Nature is Volatile -> As long as power supply available to memory we can perform Read/Write Operation on it , As soon the power supply went off content of the memory erased.
  • Faster to access(n.s)*10^-9
  • Small in Size
  • Expensive

Secondary Memory

  • it is also known as Auxiliary Memory
  • Examples -> HDD, Pen-drives, DVD
  • its Nature is Non Volatile -> As long as power supply available to memory we can perform Read/Write Operation on it , here content of the memory not gone be erased by itself Even when in no power .
  • Slower to access(m.s)*10^-3
  • Larger in size
  • Not very Expensive

As per Von-Neuman Architecture , Secondary Storage Device are part of Input / Output Devices.

Kernel (Nucleus)

kernel is main part of OS which is Core Part of OS which is responsible for managing important Hardware.

User can not have privilege to interact with Hardware Directly. User must delegate to system call API interface to access Hardware, Memory and OS routines.

- In Kernel mode , the executing code has complete and unrestricted access to the underlying hardware. it can execute any CPU instruction and reference any memory address. Kernel mode is generally reserved for the lowest-level, most trusted function of the OS . crashes in kernel mode are catastrophic , they will halt the entire PC

On Startup

When you start a PC, the program that loads first into memory is the operating system. This is because the operating system is essential for managing all other programs on the computer and is initiated by a special program called a "boot loader".

Boot process:

When you turn on your computer, the BIOS (Basic Input/Output System) searches for the boot loader program, which then loads the operating system into memory.

Boot loader:

This small program is responsible for initializing the hardware and starting the operating system.
