Flexbox Introduction

When we make any element Display flex ,then whatever the direct chidren elements present inside Flex Container they all becomes Block elements ,not matter what they are initially

flex box provides facility to align elemets along any of the one axex either Main-axies or coss axes.

Flex-Container support differet kind of properties using which we can define the allignment of flex items(Direct Children Elements).

Propeeties for Flex-Container

  • Display
  • Flex-direction
  • Flex-wrap
  • Justify-content
  • allign-item
  • allign-content
  • gap

Further Flex-item(Children Elemets) also support different kind proprties,using which we can define how the perticular Flex-Item align inside flex-container

Properties for Flex-item(Children Elemets)

  • Order
  • Flex grow
  • Flex shrik
  • Flex basis
  • allign-self

Understanding How Flex-Block Shrinks Elements

All the flex-items present in the flex container , shrink at equal percentage from its actual width ( This actual width by default is max-content - untill we not set width explicitely) to till not reaches to its minimum width which is min-content

If we would like to don`t want to shrink from any of the flex-item we can set its flex-shrink:0; by default flex-shrink is 1. We can also min-width and with property at place of flex-shrink but its not recommanded

If we have empty space in our flex-container we can use property flex-grow over flex item to cover up the space. its default value is 0

allign-item

This property over flex-container will helps us to vertically allign the flex-iems when flex direction is row , its default value is strech

Justify-content

This property over Flex-Container will hels us to horizontally allign the flex-items ,when flex direction is row

allign-item

this property over Flex-container is used to align flex lines form between wrap element(work after flex wrap)

Flex-basis

With the flex-basis property we can define , from where flex-item should be shrunken (Base Size)

Base-size of flex-items in flex-Container can be calculated from 3 ways -

  • By default Base-size is calculated from its max-width of content(If we brings all the content in a single line)
  • We can also set Base Size , by setting up width explicitely over flex item
  • Flex-Bases property also a good property for defining Base size of Flex-item.

    • Difference Between Flex-Bases AND width ?

    Flex-Bases is also a good property, One thing keep it in your mind Flex-Bases not always work along width, but its way of working is depends on in which direction we define flex-container.

    for Flex-direction->row --- work for width

    for Flex-direction->column --- work for height

    width property always be width of element not mattter where we applied it,

    By setting up Width or height of element , generally we are just making dimentions of element solidify , but Flex-bases on the other hand is just the way of setting kind of flexible dimenson, which can be grow accoding to content
