Frondend Development

Frontend development is the development of the graphical user interface of a website, through the use of HTML, CSS, and JavaScript, so that users can view and interact with that website.

  • HTML: The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.
  • CSS: The HyperText Markup Language or HTML is the standard markup language for documents designed to be displayed in a web browser.
    1. Pure CSS
    2. Bootstrap (Framework)
    3. Tailwind (Framework)
    4. Sass (Preprocessor)
  • JavaScript: JavaScript: JavaScript often abbreviated JS, is a programming language that is one of the core technologies of the World Wide Web, alongside HTML and CSS. It is use to add functionality in the website.
    1. Vanilla JavaScript
    2. jQuery
    3. React (Library)
    4. Angular (Framework)
    5. Vue (Framework)
    6. Typescript (Preprocessor)
