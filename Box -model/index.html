Important points about Box BOX-MODEL

  • In case of inline Element , it occupies width an height According to the content
  • In case of Block Element , it occupies entire available space width available but for the height it occupies as much content require space.
  • Height and width of element are depend on each other based on the content available in until we not are manipulating
  • when you set the height and width properties in CSS using percentages, the value is always calculated relative to the size of the parent element
  • Over all One thing keep it in your mind , if we try to change height or width of element using Percentages then you have to specify parent Dimension.

  • In inline element, width & height doesn't workout. width & height dependend on inner content.
  • if we add padding & border in inline element it's overral height & width increses.
  • when padding & border applied to an inline element the top and bottom padding's affect the surrounding elements.
  • In inline elements Whatever padding ,border, margin we applied they all gets applied in horizontal but in top and bottom they grow silently.
